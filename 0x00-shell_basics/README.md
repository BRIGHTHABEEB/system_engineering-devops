# Shell Basics Project

This project introduces the fundamentals of Linux file system navigation and basic shell commands.

## Project Structure

- `0-current_working_directory`: Prints the absolute path of current working directory
- `1-listit`: Displays the contents list of current directory
- `2-bring_me_home`: Changes working directory to user's home directory
- `3-listfiles`: Displays current directory contents in long format
- `4-listmorefiles`: Displays current directory contents (including hidden files) in long format

## Requirements
- All scripts must start with `#!/bin/bash`
- Each script is exactly 2 lines
- All files are executable
